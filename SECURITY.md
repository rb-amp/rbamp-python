# Security Policy

## Reporting Vulnerabilities

If you find a security issue в any rbAmp software (firmware, libraries, integrations) or hardware:

📧 **security@rbamp.com**

Please include:
- Description of the issue and impact
- Steps to reproduce
- Affected version(s) / SKU(s)
- (Optional) Suggested fix

We will:
1. Acknowledge receipt within **72 hours**
2. Investigate and respond with remediation plan within **7 days**
3. Disclose publicly only after fix is available, with credit to the reporter (if desired)

## Supported Versions

| Component | Supported version | Notes |
|---|---|---|
| Module firmware | Latest release on rbamp.com/firmware | Older versions: best-effort backport |
| ESPHome external_component | Last 2 major ESPHome releases | |
| Arduino library | Last 2 years from publish date | |

## Out of Scope

- Issues in third-party libraries (report upstream)
- Social engineering, physical access attacks
- Issues requiring root access on user's host system

## Hall of Fame

We publicly thank security researchers who responsibly disclose issues. Your name (or handle) will appear here after the issue is resolved.
